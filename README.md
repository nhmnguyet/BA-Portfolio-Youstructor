# [Project] Youstructor - Helping Yoga Teachers Manage Classes on the Go

### **Overview**
My Business Analysis Portfolio featuring the Youstructor app concept. To ensure the app actually solves the right problems, I followed a step-by-step BA process. Here’s a breakdown of my work:

* **Requirement Elicitation:** I started by drafting Q&A sheets to clarify exactly what features were needed.
  [View my Q&A Sheets](https://docs.google.com/spreadsheets/d/1Ov8LDZIPOaDEauD09Zqo1oHLxqkRR_j_/edit?usp=drive_link&ouid=117334388256851357735&rtpof=true&sd=true)

* **Documentation (BRD):** I put together a Business Requirement Document to define the project scope, including features like class registration, password recovery, and performance levels for instructors.
  [View Business Requirement Document](https://drive.google.com/file/d/152QOUISIA5vV5pG0fUMxU6YnqQpl8X-V/view)

* **Process Modeling:** I used draw.io to create BPMN diagrams for core flows like "New User Registration" and "Class Creation". I also mapped out Use Case diagrams to see how the instructor interacts with the system.
  [View BPMN & Use Case Diagrams](https://drive.google.com/file/d/1zlBd3_vTmNnTBcu13qMZSWvyNSMqdAJ6/view?usp=drive_link)

* **Wireframing & Prototyping:** I used Figma to design the mobile interface, making sure the UI is clean and easy for instructors to use on their phones.
  [Explore my Figma Mockups](https://www.figma.com/design/wK9cRslyAyhfAvGyoqsQ6w/Yostructor?node-id=0-1&t=VexklASH4aaRw9Cx-1)
