# BA-Portfolio-Youstructor
My Business Analysis Portfolio featuring the Youstructor app concept.
What I did in this project
I followed a step-by-step BA process to make sure the app actually solves the right problems. Here’s a breakdown of my work:

Requirement Elicitation: I started by drafting Q&A sheets to clarify exactly what features were needed. 
[(https://docs.google.com/spreadsheets/d/1Ov8LDZIPOaDEauD09Zqo1oHLxqkRR_j_/edit?usp=drive_link&ouid=117334388256851357735&rtpof=true&sd=true)]

Documentation (BRD): I put together a Business Requirement Document to define the project scope, including features like class registration, password recovery, and performance levels for instructors.
[(https://drive.google.com/file/d/152QOUISIA5vV5pG0fUMxU6YnqQpl8X-V/view)]

Process Modeling: To visualize how the app works, I used draw.io to create BPMN diagrams for core flows like "New User Registration" and "Class Creation". I also mapped out Use Case diagrams to see how the instructor interacts with the system.
[(https://drive.google.com/file/d/1zlBd3_vTmNnTBcu13qMZSWvyNSMqdAJ6/view?usp=drive_link)]

Wireframing & Prototyping: This was my favorite part. I used Figma to design the mobile interface, making sure the UI is clean and easy for instructors to use on their phones.
[(https://www.figma.com/design/wK9cRsIyAyhfAvGyoqsQ6w/Yostructor?node-id=0-1&t=VexkIASH4aaRw9Cx-1)]

The Tools I Used
- Analysis: Google Workspace (Docs/Sheets)
- Diagramming: Draw.io & PlantUML
- Design: Figma
